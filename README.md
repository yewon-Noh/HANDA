## 💻 프로젝트 설명

자격증 공부를 할 때 기출문제 사이트를 많이 사용한다. 매년도 문제를 풀어보고 풀이를 볼 수 있어 유용하게 사용된다. 하지만 내가 직접 문제를 질문할 수는 없다.

따라서 자격증 문제를 질문할 수 있는 커뮤니티 플랫폼을 구현하고자 하였다.

## ⚙ HOW TO USE

NPM 모듈 설치

```
npm install
```

실행

```
npm start
```

Open http://localhost:3000 in browser

## 🔧 프로젝트 구조

```
HANDA
⊢ public
⊢ ⊢ css         // FE css 파일
⊢ ⊢ images      // 이미지 파일
⊢ ⊢ js          // BE route 파일
⊢ package.json
⊢ package-lock.json
⊢ README.md
⊢ menual  // 설정 메뉴얼 정리
```
